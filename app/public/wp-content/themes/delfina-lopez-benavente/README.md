# Delfina López Benavente - Tema hijo

Tema hijo de Blocksy para la web profesional de Delfina López Benavente, Estratega de Contenido Digital.

## Instalación

1. Activa el tema **Delfina López Benavente** en Apariencia > Temas
2. Ve a **Apariencia > DLB Setup** para configurar la web

## Setup (Apariencia > DLB Setup)

1. **Importar Header y Footer** - Crea las plantillas de cabecera y pie
2. **Crear página de inicio** - Crea la página principal para editar con Elementor
3. **Crear formulario de contacto** - Crea el formulario WPForms con todos los campos
4. **Instagram Feed** - Enlace para instalar el plugin de Instagram

## Estructura de la página de inicio

Edita la página de inicio con Elementor y añade las 8 secciones (usa los IDs para el menú de anclas):

- `#hero` - Hero
- `#servicios` - Servicios
- `#como-trabajo` - Cómo trabajo
- `#proyectos` - Proyectos
- `#fotografia` - Fotografía
- `#instagram` - Contenido reciente (Instagram)
- `#sobre-mi` - Sobre mí
- `#contacto` - Contacto

El archivo `CONTENIDO-SECCIONES.md` contiene todo el texto para cada sección.

## Menú de navegación

Crea un menú con enlaces de ancla:
- Servicios → #servicios
- Cómo trabajo → #como-trabajo
- Proyectos → #proyectos
- Fotografía → #fotografia
- Sobre mí → #sobre-mi
- Contacto → #contacto
